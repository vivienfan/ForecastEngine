This program has very specific input and output that can only be used by Siemens Building Technology sales department.
The application has two functionality, to Update individual forecast by different salesman into the main database, or to look up specific forecast data in the main database.
To update, it takes in the directory of indicidual forecast and the directory of the main database as input.
And as a result, it updates the main database, mark cases as delayed, on-time, ahead, closed if necessary.
To lookup, it takes in salemans'name, mark option, and directory of the main database as input.
And as a result, it outputs the information of all matched cases into a new Excel file. 
As it operates, it proves a window of processing tasks to inform user the progress.
